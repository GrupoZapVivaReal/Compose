ViewUnit

public struct ViewUnit<View: UIView>: TypedUnit, SelectableUnit

Generic Unit, that can encapsulate any UIView inside it. It uses ComposingUnitCollectionViewCell as its CollectionViewCell if the inner view is not a UICollectionViewCell subclass. You can use it as : let unit = ViewUnit<UILabel>() { label in }
